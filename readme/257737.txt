README

Configuration

Indicate the host, user, database, and port via the usual PostgreSQL environment variables:
- PGHOST
- PGUSER
- PGPORT
- PGDATABASE

Set the password using ~/.pgpass, per the documentation:
   <http://www.postgresql.org/docs/9.1/static/libpq-pgpass.html>

Project Description:
A python script that uses ncurses to display an overview of various info about a schemaverse (https://schemaverse.com/) account
