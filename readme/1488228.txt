stretchmesh
===========


# Compiling

To compile, ensure that the `$MAYA_VERSION` environment variable is set. To compile Maya for 2013, set it to `2013`:

    export MAYA_VERSION=2013

Then, run `make`, and `stretchMesh2013.bundle` should have been created.
