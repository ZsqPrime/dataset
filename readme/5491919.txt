Project Description:
Adds page into more than 1 category

README:
Multicat
========

Добавляет возможность добавлять страницу в несколько категорий

Установка
---------

В файле **page.add.tpl** тег:

    {PAGEADD_FORM_MULTICAT}

В файле **page.edit.tpl** тег:

    {PAGEEDIT_FORM_MULTICAT};

**page.tpl**: 

    {PAGE_MULTICAT}
