Project Description:
Process real-time transit GPS data into a more usable forms: Discovers Routes, Predicts Arrivals.

README:
###########
  Caravel
###########

Caravel reads and analyses transit GPS coordinates to discover routes and
predict next bus arrival times.

This is a Python 2.7 project. http://www.python.org

It depends on Sphinx to produce the full suite of documentation.
http://sphinx.pocoo.org
