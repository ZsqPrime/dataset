#jSelectMulti

jQuery plugin to progressively enhance HTML <select multiple='multiple'> widgets into a user-friendly dual-pannel interface.

##Usage
```js
$("select[multiple='multiple'][name='ingredients']")
.jSelectMulti({
    'add_btn_text': 'Add to list, won\'t you?'
});
```
