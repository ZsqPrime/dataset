README:
INTERLINEAR SCRIPTURE ONLINE

Este sistema tem o intuito de oportunizar a análize online das escrituras sagradas nos seus originais hebraico e grego
intelinearmente com as traduções para as liguas inglesa e portuguesa. 

Este projeto utiliza-se de tecnologias livres ou apoiadas sob a licença GPL, sendo o mesmo também apoiado por esta 
licença.
