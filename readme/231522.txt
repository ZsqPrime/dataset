pip2spec
========

This project aims at providing an easy way to generate a spec file
for projects hosted on pypi:
http://pypi.python.org/pypi

Dependencies:
------------
- python-argparse
- python-jinja2


Project Description:
Makes your life easier to generate Fedora spec file for pypi project
