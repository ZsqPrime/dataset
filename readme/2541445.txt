Project Description:
Now you can use the power of wikis for taking notes for all your papers. This plugin creates wiki pages for all your papers in your Mendeley account and keeps track of the meta data.

README:

Download Mendeley Open API Python Client
========================================

Download mendeley_client.py from following repository:
https://github.com/Mendeley/mendeley-oapi-example

Installation as Desktop Version
===============================

Download and unpack MoinMoin from here:
http://moinmo.in/MoinMoinDownload

Copy Mendeley2Moin.py and mendeley_client.py to 
moin-1.9.X/wiki/data/plugin/action/ . Run wikiserver.py (see MoinMoin README 
for details). Setup user account and import system pages. Click "more actions" 
and choose "Mendeley2Moin". Follow the installation steps.
