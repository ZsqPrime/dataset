Project Description:
The Kunstmaan sandbox is an example project to demonstrate what the Kunstmaan bundles are able to achieve when using a combination of the Kunstmaan bundles.

# Kunstmaan Sandbox

This is a sandbox containing [Symfony Standard edition](https://github.com/symfony/symfony-standard) with the [Kunstmaan bundles](http://bundles.kunstmaan.be).
