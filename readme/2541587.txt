Project Description:
projeto em LaTeX de e-book em quadrinhos

README:
DESCRICAO

Livro e-book de tiras em quadrinhos de uma familia brasiliense em minas gerais.

COMO COMPILAR?

1. Baixe o codigo fonte deste projeto

2. instale o LaTeX

3. Execute>

> cd hq
> latex hqfamilia.tex
> dvipdfm hqfamilia.dvi

4. Sera gerado o arquivo "hqfamilia.pdf" no diretorio /hq.
