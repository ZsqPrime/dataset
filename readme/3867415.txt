Project Description:
Google Closure Compiler for Sublime Text 2/3 Build Systems (Windows,Mac OS X, Linux)

README:

Open a .js file, press F7 to build width Google Closure Compiler.

Messages will show in the Console panel.
When an error message appears, you can double-click it to jump the file and line.
