# depthkit.js

simple 3d engine for the canvas 2d context (so no webgl)

It does not have all the features of a real engine, but it still can do a number of decent things:

- movable camera
- parse obj files
- lights
- fog

# depthkit.js

simple 3d engine for the canvas 2d context (so no webgl)

It does not have all the features of a real engine, but it still can do a number of decent things:

- movable camera
- parse obj files
- lights
- fog
