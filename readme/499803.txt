MyFormBuilder
=============

MyFormBuilder is a small JavaScript app to generate forms by a wufoo-like
interface.

This app depends on:

* jQuery
* knockoutjs
* jQueryUI
* twitter bootstrap
* jQuery hotkeys

All dependent libraries and stylesheets are stored in `lib` directory.



Project Description:
Form builder in javascript
