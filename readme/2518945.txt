Project Description:
iOS InAppEmail Class

README:
﻿Sending an email from your iPhone application

Sending emails from within an iOS application is not difficult. In addition, with  introduction of Message UI Framework in iOS 3 we can send in-app messages without leaving the app greatly improving user experience.

I’d like to simplify sending emails even further. The InAppEmail  class you can download from GitHub  will limit required effort to just few clicks and 2 lines of code.

More details can be found here:
http://spritebandits.wordpress.com/2012/01/09/sending-an-email-from-your-iphone-application/
