Project Description:
A tool for copy and write backup of the Xbox 360

README:
Xbox360Burner è uno script che rende più semplice la creazione di un iso e masterizzazione dei giochi per Xbox 360.
Supporta anche giochi XDG3.

Per installare lo script basta dare i permessi di esecuzione al file "install" ed eseguirlo nel terminale
