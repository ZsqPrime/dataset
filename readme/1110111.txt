## Prerequisites

	nodejs
	npm

## Install

	$ git clone git@github.com:csuwldcat/WebBuilder.git
	$ git submodule init
	$ git submodule update
	$ npm install	
	$ cd vendor/shipyard	
	$ npm link
	
	install addon/web-builder-bootstrap.xpi into Firefox
