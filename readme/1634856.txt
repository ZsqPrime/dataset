README
======
This Plugin adds an "Upload to imgur" menu item in the diodon menu whenever an image is selected in the clipboard

![screenshot](https://github.com/LeonB/diodon-imgur/raw/master/screenshot.png)

DEPENDENCIES
------------
* gir1.2-diodon-1.0
