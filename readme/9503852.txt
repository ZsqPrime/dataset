Project Description:
FreeCell for CLI is a complete suite of card games which can be run on any 80x25 wide terminal. You name it: DOS, Win32, Linux, OS/2 - anything that has some version of curses library. You can play it over LAN. or you can have a Qt based GUI.

README:

# FreeCell for Linux
**NOTE:**

This program is derived from FreeCell for DOS v 1.1 by me. It was for Turbo C++ v 3.0, and used custom libraries. Most of such code is removed and aim of this program is to just be compatible with g++. The original program was not throughly checked for bugs.
