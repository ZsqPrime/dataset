README:
wallchanger:
Nick Lange

Change your wallpaper on a Linux machine using gconftool.
Useful to put in a cron job to run every X minutes (or whenever you like).

To use:
Edit the dir variable to point at your directory of images.
For now, only PNGs and JPGs are accepted.
