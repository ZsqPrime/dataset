Project Description:
A tool for managing the Daisy Pipeline's TTSBuilder configuration file. Define which TTS engine must be used for which locale.

README:
Dependencies:
=============

 + xalan: XML transformations

 Windows:
 --------
 + sapi: Java bindings for Microsoft Speech API
    + com4j: Java-COM interface
 + okapi: Support for LCID language codes

 Mac:
 ----
 + rococoa: Java-Cocoa interface
    + jna: Java Native Access
    + slf
    + cglib
       + asm
