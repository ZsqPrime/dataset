Assembly Reference Lister
=========================

A console tool to quickly print out all references and dependencies of an assembly.

Usage
-----

    Usage:
        arl.exe <NameOfAssembly|PathToAssembly> (Options)

    Options:
        --with-gac          # will include assemblies from GAC

Project Description:
Lists all references and its dependencies of a given assembly
