#### Installation

```
git clone https://github.com/fieldville/dotvim.git ~/.vim
```

##### Create symlinks

```
ln -s ~/.vim/vimrc ~/.vimrc
ln -s ~/.vim/gvimrc ~/.gvimrc
```

then launch vim!
