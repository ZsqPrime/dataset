A standalone program that packages Django web framework for easy testing. This can also be used as a basis for packaging a Django project into standalone cross-platform program.

Project Description:
A self contained Django web framework launcher. It is cross-platform but the current build is only for Mac. Trying out Django in the Mac is now as easy as dropping the app to your Applications folder, then launch. Easy! ;)
