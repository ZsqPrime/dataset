== Clipperz Widget ==
Integrates the Clipperz password manager into your Firefox browser.  Enabled auto-login and auto-paste for your passwords stored in the Clipperz database.

Visit http://www.danielmorlock.de/2011/clipperz-widget for more information.


Project Description:
An addon to integrate the Clipperz Communicy Edition into Mozilla Firefox.
